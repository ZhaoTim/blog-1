- 实现动画的几种方案：
  - CSS 动画	
  - SVG 动画
  - Canvas 动画

- 动画方法
    - 补间动画/关键帧动画
    - 逐帧动画
    - 变换动画（平移、旋转、缩放、斜歪、矩阵变换）
    - 变形动画（描边、路径轨迹、路径变形）
    - 过渡动画
    - 蒙版动画
    - 填充动画
    - 遮罩动画

- 动画开发时需要注意的问题：
  - 动画出现的顺序
  - 惯性	
  - 透视（近大远小）
